My passion has always lied in the technology industry, which has brought me here today. Down below you can see breif summary of what I can offer.

Data Collection/Wrangling

Data can get messy, one of the first steps of data analysis is just preparing the data for analysis. That’s where I come in to collect and clean it.

Data Analysis

Using the right statistics and data tools, I can conduct thorough analysis. Squeezing as much information as possible from the data. Finding those actionable insights to help drive your business forward.

Data Visualization

Now comes my favourite part, displaying the insights. In a clear and concise dashboard, interactive and beautiful. Easy to understand, easy to use.

Projects

Adidas Dashboard: This is my favourite project, hence it's at the top. Utilzing Power BI, I made a multi-paged dashboard containing Sales, Customer and HR data. It’s completely interactive. Meaning it has slicers that filter the data, search bar options to find certain employees / customers and page navigation buttons.


Car Sales Report: More of a fun project. This beautiful report was done in Excel. My main goal was to see how far I could push the Excel formatting whilst still conveying useful information on the report. I started by cleaning the data and creating all my visuals, then I combined it all together.


Maven Pizza Analysis: In this project I took on the role as the lead Data Analyst working for a pizza place called Maven Pizza. My task was to answer a select amount of questions from the owner. To answer these question I had to analyze the data set provided and retrieve insights from it, using my skills on SQL and Excel. This included running multiple queries on SQL , therefore the data could be manipulated to give me answers to the questions asked. Then I proceeded to use Excel to create a Dashboard that I could present to the owner. Check out my GitHub at the bottom of the website if you wish to see the SQL script.


Life Expectancy Dashboard: Using Tableau I analysed and visualized a life expectancy data set. I wanted to keep this dashboard clear and concise, easy for anyone to understand. Also giving the option for the user to interact with the data and filter it.


LA Crime Data Dashboard: In this project I started with cleaning the raw data in Excel, then I imported the data into SQL so I could run some queries on it and find out what type of data I was working with. Next, I transferred the main data set and my custom SQL queries into Tableau, I did some further calculations and then I created a dashboard including the insights I found and my suggestions. The SQL script is on my GitHub.


Balaji Fast Food Sales: This project was the turning point where I realized that to truly create something beautiful with Power BI, you must outsource graphical elements of a dashboard. Therefore, I utilized another Microsoft platform, PowerPoint. Combining the design of PowerPoint and the analysis from Power BI, I created an interactive Sales Report. Not only is it aesthetically pleasing, it’s insightful.


Animal Dashboard: This was another fun project. Referring to the Balaji Fast food Sales, I once again used PowerPoint to make this dashboard stunning. After cleaning the data in Excel I transported it to Power BI and analysed the data. The data set consists of different animals and statistics related to them. My main idea was to create a slicer where you can select any animal and retrieve all the details about it. Furthermore, it contains data displaying the Top / Bottom figures of each animal before using the slicer..
